构建工具，依赖管理

maven排在Java生态的第一，Gradle用于Android

基于XML格式的文件描述依赖配置

  • https://maven.apache.org/pom.html#What_is_the_POM
  • POM（Project Object Model）
  • pom.xml
    • groupId,artifactId,version - 坐标
    • type - 类型
    • scope - 范围
    • optional - 依赖是否可选
    • exclusions - 排除传递性依赖

依赖仲裁原则

  • 最短路径优先
  • 第一声明有限

Best Practice

  • 减少重复配置的代码
  • 减少不确定因素

问题解决

  • 确认系统，确认Java，Maven版本
  • 确认本地依赖的缓存
  • 检查完整依赖树
