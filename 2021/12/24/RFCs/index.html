What is an RFC?

  • Request for comments
  • 目的是为新feature进入框架提供一个一致，可控的路径
  • pull request 能搞定很多changes，bug fixes等，但是一些 “substantial” - 实质性的改变，这些是需要经过一定的设计过程，并能在Vue的核心团队和社区中达成共识的
  • Substantial的判断标准
